## 📌 About The Project

WanderHigh is a fictional Himalayan trekking agency website created as a frontend portfolio project. It showcases real-world UI/UX patterns including dynamic content rendering, filtering, animations, and multi-page navigation — all built without any framework.

---

## ✨ Features

- 🎨 **Modern UI Design** — Clean, professional design with custom color palette and typography
- 🔍 **Filter Treks** — Filter by difficulty (Easy / Moderate / Hard) using vanilla JS
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet and desktop
- 🎞️ **Scroll Animations** — Elements animate in using Intersection Observer API
- 🎠 **Testimonial Slider** — Auto-advancing slider with dot navigation
- 📋 **Dynamic Trek Detail Pages** — Each trek loads its own data from a central JS data file
- ✅ **Form Validation** — Contact and newsletter forms with JS validation
- 💾 **LocalStorage** — Newsletter subscriptions saved in browser storage
- 🗺️ **Multi-page Navigation** — Home, About, Contact, Trek Detail pages

---

## 🛠️ Built With

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Styling, Flexbox, Grid, Animations |
| JavaScript (ES6+) | Interactivity, DOM manipulation, LocalStorage |
| Google Fonts | Playfair Display & DM Sans typography |
| Intersection Observer API | Scroll reveal animations |

## 📁 Project Structure

```
wanderhigh/
├── index.html              # Homepage
├── css/
│   ├── style.css           # Main styles
│   └── pages.css           # Inner page styles
├── js/
│   ├── data.js             # All trek data (easy to add more)
│   └── main.js             # All interactivity
├── images/                 # Trek images
└── pages/
    ├── about.html          # About page
    ├── contact.html        # Contact page
    └── trek-detail.html    # Dynamic trek detail page
```

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/Nikhilyadav12889/wanderhigh.git
```

2. Open the project folder in VS Code

3. Install **Live Server** extension in VS Code

4. Right-click `index.html` → **Open with Live Server**
